# 🧠 Mapas Mentales

Coloca aquí las imágenes PNG de los mapas mentales para cada integrante.

## Formato esperado (obligatorio):

```
MapaMental.tema1.png    → Mapa Mental de Rafael García
MapaMental.tema2.png    → Mapa Mental de Lorena Gomez
MapaMental.tema3.png    → Mapa Mental de Nicolás García
MapaMental.tema4.png    → Mapa Mental de Facundo Domínguez
MapaMental.tema5.png    → Mapa Mental de Valentin Fernández
MapaMental.tema6.png    → Mapa Mental de Santiago Carrillo
```

⚠️ **Importante**: 
- Los nombres deben ser exactamente como se muestran arriba (respetando mayúsculas y puntos).
- Los archivos DEBEN ser en formato `.png` (las imágenes en PNG reemplazarán a los PDFs que había previamente).

Estos mapas serán visualizables desde la sección "Guiones" con un botón "Ver Mapa Mental". Si ya tienes los PNG listos, sube los archivos con los nombres indicados y reemplazarán automáticamente los anteriores.
