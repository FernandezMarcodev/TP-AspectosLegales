# 📊 Presentaciones

Coloca aquí las presentaciones en PDF para cada integrante.

## Formato esperado:

```
Presentacion.tema1.pdf    → Presentación de Rafael García
Presentacion.tema2.pdf    → Presentación de Lorena Gomez
Presentacion.tema3.pdf    → Presentación de Nicolás García
Presentacion.tema4.pdf    → Presentación de Facundo Domínguez
Presentacion.tema5.pdf    → Presentación de Valentin Fernández
Presentacion.tema6.pdf    → Presentación de Santiago Carrillo
```

⚠️ **Importante**: Los nombres deben ser exactamente como se muestran arriba (respetando mayúsculas y puntos).

Estas presentaciones se visualizarán de forma **concatenada** en la sección "Presentación", dando una experiencia continua de navegación entre todas ellas.
